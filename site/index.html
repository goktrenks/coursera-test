About the usage of "<",">" and "&" in HTML. They can't be written directly, otherwise the browser won't interpret them correctly. These are called HTML entities.
  • & = "&amp;"
  • > = "&gt;"
  • < = "&lt;"
If you want to make sure some text is always together on the same line as the screen size is decreased, type "&nbsp;" in the place where you don't want the browser to break your sentence. Please note that this also acts as a space, so pay attention to that.
